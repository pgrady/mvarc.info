Mt. Vernon Amateur Radio Club Volunteer Examiners

We administer amateur radio examinations monthly near downtown Alexandria, Virginia.

Bring to the exam:
* Picture ID issued by a Government agency
* FRN if you're currently licensed or if you've registered with the FCC (Social Security Number must be provided if you have not received an FRN)
